# AI Codebase Navigator

React + Vite UI for the Codebase Navigator backend. Connect a GitHub repo, browse files, and ask questions (SSE streaming).

## Run

1. Start the API (e.g. `http://localhost:8000`).
2. `npm install`
3. `npm run dev` — open the URL shown (default port 8080).

Set **API Base URL**, **repository URL**, and **API token** in the top bar, then **Connect**.

## Build

```bash
npm run build
```
